# 💳 Credit Card Fraud Detection Using Big Data Analytics with Apache Spark

## 📌 Project Overview

This project focuses on detecting fraudulent credit card transactions using Big Data Analytics techniques with Apache Spark. The system processes large-scale transaction datasets efficiently and provides interactive visualizations through a Streamlit web application.

The project demonstrates the use of distributed data processing, fraud analytics, data visualization, and interactive dashboard development for financial cybersecurity applications.

---

# 🚀 Features

- Upload credit card transaction dataset
- Analyze fraudulent and legitimate transactions
- Big data processing using Apache Spark
- Interactive dashboard using Streamlit
- Fraud distribution analysis
- Transaction amount analysis
- Correlation heatmap visualization
- High-value fraud transaction detection
- Dark-themed professional UI

---

# 📂 Dataset

The project uses a Credit Card Fraud Detection dataset containing:

- Transaction features
- Transaction amount
- Fraud classification labels

Class Labels:
- `0` → Legitimate Transaction
- `1` → Fraudulent Transaction

---

# 🧠 Problem Statement

Traditional fraud detection systems struggle to process massive financial transaction data efficiently. With the rapid growth of online banking and digital payments, scalable big data analytics solutions are required to identify fraudulent transactions accurately and quickly.

---

# ✅ Proposed Solution

This project uses Apache Spark for distributed data processing and Streamlit for interactive visualization. The system analyzes transaction patterns, detects fraud trends, and presents insights through a web-based dashboard.
